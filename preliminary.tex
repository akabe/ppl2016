\begin{block}{準備：条件付き確率場 {\small{}[Lafferty ICML2001]}}
  \begin{itemize}
  \item 確率変数の間の複雑な依存関係をグラフで表現
  \item 自然言語処理などで用いられる確率モデル
  \end{itemize}
  \structure{具体例}：``John loves Alice'' の品詞予測
  \begin{itemize}
  \item 因子ノード（黒い四角）ごとに点数を計算
  \item \alert{合計点数が最大}となる品詞の割り当て方が正解\\（の可能性が高い）
  \end{itemize}
